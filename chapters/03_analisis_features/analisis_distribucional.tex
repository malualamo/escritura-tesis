\section{Análisis distribucional}
\label{sec:analisis_distribucional}

% Distribuciones de cada feature con estadísticos descriptivos
% (media, mediana, IQR, outliers).
%
% Para cada feature: histograma o boxplot, media, mediana, desviación
% estándar, percentiles 5 y 95. Identificar features con distribución
% degenerada (e.g. masa puntual, valores constantes por trial).
%
% Analizar por separado features estáticas vs. dinámicas y, dentro
% de las dinámicas, la evolución a lo largo del trial (posición de
% fijación 1, 2, ..., k).
%
% Señalar features candidatas a transformación (log, normalización)
% antes del modelado.

A continuación se presenta el análisis distribucional de las features 
extraídas del modelo nnELM, calculadas sobre las 61.252 fijaciones del 
dataset. Para cada grupo de features se reportan estadísticos descriptivos 
y la evolución del valor promedio a lo largo del trial, diferenciando entre 
trials en los que el sujeto encontró el target y trials en los que no. Las 
features \texttt{H\_prior\_sal} y \texttt{H\_prior\_mod} son estáticas 
(constantes por trial) y se analizan por separado. Las features 
\texttt{gan\_informacion}, \texttt{eig\_fij}, \texttt{gap\_entr}, 
\texttt{dist\_gri} y \texttt{dist\_pix} presentan $N = 56.751$ observaciones 
dado que requieren el mapa de entropías esperadas del paso $t+1$, que no 
existe para la última fijación de cada trial.
